# AstrBot Sudoku Plugin

功能：
- 生成唯一解数独题目（简单/中等/困难）
- 对局持久化与超时清理
- 图片渲染展示棋盘（优先使用PIL本地渲染）
- 填错扣生命值（默认 3 次，归零结束）

安装：
1. 复制插件目录到 `AstrBot/data/plugins/astrbot_plugin_sudoku`
2. 在 AstrBot WebUI 启用插件
3. 根据需要修改配置（字体路径、渲染方式、生命值等）

使用：
- `/数独 简单|中等|困难` 开始新游戏
- `/数独 查看` 查看当前棋盘
- `/数独 填 A1 5` 或 `#数独 a11` 快速填数
- `/数独 结束` 结束当前游戏

指令：
- /数独 [简单/中等/困难] 开始新游戏
- /数独 查看
- /数独 填 A1 5
- #数独 a15
- /数独 结束

配置：见 `_conf_schema.json`

说明：
- 插件内置中文字体 `assets/LXGWWenKai-Regular.ttf`（约 23.5MB），用于解决中文渲染缺字问题。
- 如需减小体积，可在配置中将 `font_path` 指向系统字体或自行删除字体文件并改用 HTML 渲染。
